# archibold.io

Options:

  * **install** software via `bash <(curl -s archibold.io/install/$software)`
  * **require** utils via `source <(curl -s archibold.io/require)`

Following an example:

```
source <(curl -s archibold.io/require)
require echomd
require clip

echomd "Copying to clipboard your name *$USER*"
echo $USER | clip
```

To know the **list** of about available installers or utilities:

  * `bash <(curl -Ls archibold.io/install)`
  * `bash <(curl -Ls archibold.io/utils)`

Please note the `-Ls` command in list cases is mandatory.

'
